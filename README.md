# AURA: 교실 속 난청 아동을 위한 지능형 청취 보조 시스템

## 1. 프로젝트 소개

**AURA**는 교실 내 심한 소음과 잔향 환경에서 **난청 아동이 교사의 음성을 더 선명하게 들을 수 있도록 돕는 지능형 청취 보조 시스템**이다.  
기존 보청기·로저/FM 시스템이 가진 한계를 보완하고, **엣지 디바이스 상의 실시간 음성 처리**를 통해 난청 아동의 학습 접근성을 높이는 것을 목표로 한다.

- 대상: 소음이 심한 일반 교실 환경의 난청 아동
- 형태: 책상 위에 두는 보조기기 + 이어폰(또는 보청기 입력) 연동
- 핵심 기능: 교사 음성 중심의 **목표 화자 분리(Target Speaker Extraction, TSE)** 및 소음 억제

---

## 2. 문제 배경

일반 교실에서는 다음과 같은 요인으로 인해 난청 아동의 어음 명료도가 크게 떨어진다.

- 평균 **50 dB 이상**의 배경 소음 (학생 잡담, 의자·책상 소리 등)
- **0.6–0.9초 수준의 잔향 시간**(RT60)로 인한 음절 겹침
- 교사-학생 간 거리 증가에 따른 신호대잡음비(SNR) 악화

기존 보청기와 로저/FM 시스템은
- 보청기는 **단순 증폭 중심**이라 소음 속에서 특정 화자만 골라내기 어렵고,
- 로저/FM 시스템은 교사가 마이크·송신기를 별도로 착용해야 하는 불편함과 설치 비용의 한계를 가진다.
- 최신 AI칩을 탑재한 보청기들은 이를 보완하여 소음환경에서 일정수준?의 청취환경을 제공하지만, ?? (phonak사의 DEEPSONIC을 활용한 보청기 만족도가 어느정도인지 잘 모르겠음)

AURA는 **교사가 별도 마이크를 착용하지 않아도**, 교실 내 특정 방향(칠판/교탁)의 음성을 선택적으로 강화하는 것을 목표로 한다.

---

## 3. 시스템 개요

### 3.1 목표

- 엣지 디바이스(Jetson, Xavier 등)에서
  - **실시간(지연 200 ms 이하)**으로
  - 목표 화자(교사) 음성을 분리하고,
  - 주변 소음 및 잔향을 줄여
- 난청 아동에게 이어폰 또는 보청기 입력 단자를 통해 **더 깨끗한 음성 신호**를 전달하는 것.

### 3.2 주요 기능

- **목표 화자 분리 (Target Speaker Extraction, TSE)**
  - 잡음·다중 화자 환경에서 교사 음성만을 분리
- **소음 억제 (Noise Suppression) & 디리버브 (Dereverberation)**
  - 교실 배경 소음 및 잔향 감소
- **지향성 마이크 입력 처리**
  - 칠판/교탁 방향의 신호를 우선적으로 수집
- **온디바이스(온-엣지) 처리**
  - 개인정보 보호 및 네트워크 의존도 최소화
