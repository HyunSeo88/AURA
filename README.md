# AURA: 교실 속 난청 아동을 위한 지능형 청취 보조 시스템

## 1. 프로젝트 소개

**AURA**는 교실 내 심한 소음과 잔향 환경에서 **난청 아동이 교사의 음성을 더 선명하게 들을 수 있도록 돕는 지능형 청취 보조 시스템**이다.  
기존 보청기·로저/FM 시스템이 가진 한계를 보완하고, **엣지 디바이스 상의 실시간 음성 처리**를 통해 난청 아동의 학습 접근성을 높이는 것을 목표로 한다.

- 대상: 소음이 심한 일반 교실 환경의 난청 아동
- 형태: 책상 위에 두는 보조기기 + 이어폰(또는 보청기 입력) 연동
- 핵심 기능: 교사 음성 중심의 **목표 화자 분리(Target Speaker Extraction, TSE)** 및 소음 억제
